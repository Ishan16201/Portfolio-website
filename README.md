# Portfolio-website
Ishan Saraf: Quantitative Trader, Author, and Researcher Portfolio

This repository hosts the official personal portfolio website for Ishan Saraf, a quantitative trader, author, and researcher specializing in market liquidity, algorithmic trading systems, and macroeconomic analysis.

The website is a single-page application designed for speed, responsiveness, and a professional, dark-themed aesthetic using modern web technologies.

This website is a static site and can be deployed instantly using several services.

Recommended Methods:

GitHub Pages (Easiest):

Ensure the index.html file is in the root directory of your repository.

Go to your repository Settings -> Pages.

Select the main branch (or your primary branch) and the / (root) folder.

Your site will be live at yourusername.github.io/your-repo-name.

Netlify / Vercel:

Connect your GitHub account.

Select the repository and click "Deploy Site." The build process is automatically detected and requires zero configuration.

Local Setup

To view and edit the site locally:

Clone this repository (or copy the index.html file).

Open the index.html file directly in any web browser. No local server is required.
